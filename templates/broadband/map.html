Google Spreadsheet API, Python, Google Charts API Example

This is a small example of using the google spreadsheet API and the google charts API to do a small, quick and easy data hack. We've used the Guardian Data Store for some data in this example (specifically the OECD Broadband Data), but there's no reason why you couldn't slightly adjust the code and use it on your own data.
